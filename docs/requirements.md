# 要件定義書

（プロジェクト名：devstep-daily-report）

## 1. サービス概要

本サービスは、チームで仕事を行うユーザー向けの日報管理アプリです。

## 2. ターゲットユーザー

- 小規模なチームのマネージャーとそのメンバーがターゲットユーザーです。

## 3. 解決する課題

| 課題                               | 解決策                           |
| ---------------------------------- | -------------------------------- |
| 日報をExcelで管理している          | Webで一元管理                    |
| チームメンバーの業務内容が見えない | 全員の日報が一覧できる・見える化 |

## 4. 必須機能一覧

ログイン、サインアップ
日報作成
日報一覧
日報編集
日報削除
プロフィール編集

## 5. 非機能要件

スマホのブラウザからの操作

## 6. 実装しない機能（スコープ外）

日報の検索・フィルタ機能
コメント機能
パスワードリセット
